### 📝 Jemi Project: Phase 2 개발 재개 프롬프트

**[프로젝트 현황 요약]**

1. **UI/UX:** v5.1 다크 테마 IDE 기반. 사이드바는 Dashboard, Project Management, Mark down(하이어라키 구조)으로 분류 완료. (테마 및 폰트 고정 지침 유지)
2. **아키텍처:** [Ingestion] → [Analysis] → [Decision] → [Report] 4단계 선형 구조 확정.
3. **핵심 강조:** Jemi 모델(AI01~03) 및 Decision Layer는 **노란색 네온(Neon Yellow)** 스타일로 시각적 정체성 부여.
4. **결정 로직:** Decision Layer 내 **AI03(Dynamic Weight Tuner)**이 모든 분석 모듈(M01~M04, AI01~02)의 결과값을 취합하여 최종 가중치를 할당하는 중앙 통제 방식 채택.

**[다음 작업 목표]**

* **Analysis Layer 실전 구현:** `Multi-dimensional Analysis Model` 내 첫 번째 엔진인 **M01(Trading Subject Model)**의 수학적 알고리즘 설계.
* **Decision Layer 구체화:** AI03이 각 도메인 점수를 어떤 '가중치 규칙 테이블'에 의해 튜닝할지 **`docs/logic_specs.md`** 정의 시작.
* **Jemi 개입 수준 설정:** 숫자의 신뢰도를 검증하는 '감찰관'으로서의 AI 프롬프트 고도화.
